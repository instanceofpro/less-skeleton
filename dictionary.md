Словарь сокращений
=============

Данный словарь не является обязательным к использованию. Полное слово всегда читебельнее, чем его сокращение. Но если нужно что-то сократить, то лучше это делать в рамках словаря.

* `btn` - button
* `chk` - checkbox
* `ctr` - container
* `ftr` - footer
* `hdr` - header
* `icn` - icon
* `img` - image
* `itm` - item
* `lbl` - label
* `lgc` - logic
* `lnk` - link
* `lrg` - large
* `lst` - list
* `mdm` - medium
* `msg` - message
* `plg` - plugin
* `plh` - placeholder
* `psw` - password
* `slc` - select
* `sml` - small
* `ttl` - title
* `txr` - textarea
* `txt` - text

Сокращение должно состоять из 3-х букв.
